# Forensic Use Cases

## Cybersecurity
- Detect tampering in critical datasets and verify evidence integrity.
- Export SIEM‑ready logs for incident response.
- Produce evidence bundles for audit and compliance.

## Data Integrity
- Prove step‑by‑step transformations for regulated pipelines.
- Trace schema drift and content churn across steps.

## AI/ML Pipelines
- Verify feature set integrity between training and inference runs.
- Capture dataset drift with row‑level diffs.

## IT / Platform Engineering
- Track pipeline changes across environments.
- Integrate with Airflow/Dagster/dbt and centralize audit evidence.
